# Roadmap — Dois Mundos

## Milestone 1: Core Engine & Movement (Foundation)

- **F01 — Game Loop & Clock**: Game time system, day/night cycle, schedule constraints
- **F02 — Isometric World & Movement**: Tile-based grid, keyboard movement (arrows), collision system
- **F03 — Character System**: Two characters (Rosa/Azul), needs bars, expressions, switchable control

## Milestone 2: World Building

- **F04 — Favela Home**: Small environment, low-quality objects (1-2 stars), basic furniture
- **F05 — Mansion Home**: Large environment, high-quality objects (4-5 stars), luxury furniture
- **F06 — School**: Shared environment, medium-quality objects (2-3 stars), classrooms, cafeteria

## Milestone 3: Interaction & Mechanics

- **F07 — Object Interaction System**: Enter key interaction, quality multipliers, action menus
- **F08 — Inequality Mechanics**: Compound penalties, commute simulation, energy/hunger cascading effects
- **F09 — NPC System**: Brighta teacher, basic character-to-character interaction

## Milestone 4: Progression & Education

- **F10 — Daily Mission System**: 10 daily missions, SAT bonus, completion tracking
- **F11 — SAT Quiz Integration**: Real questions from CollegeBoard, triggered on study/class actions
- **F12 — College Progress Bar**: University admission progress, score tracking, end-game condition

## Milestone 5: UI & Polish

- **F13 — HUD & Interface**: Needs bars, clock, mission panel, character switch UI
- **F14 — Expressions & Animations**: 6 expression states, floating icons, commute animations
- **F15 — Visual Polish**: Lighting, shadows, glow effects, consistent pixel art style

## Milestone 6: Web Deploy

- **F16 — Godot Web Export**: HTML5 build, performance optimization, browser compatibility
